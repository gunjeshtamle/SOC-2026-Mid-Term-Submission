The Markov property states that the future state of a system depends only on the current state and the current action taken, entirely ignoring the sequence of events that led up to it.  
Mathematically, it means the probability of the next state, given the present state and action, is identical to the probability given the entire history of past states and actions.  

### Significance in RL
A Markov Decision Process (MDP) provides a formal framework to model sequential decision-making in Reinforcement Learning. It defines how an agent interacts with an environment through states, actions and rewards to learn optimal behavior over time.  
A Markov Decision Process combines states, actions, transition dynamics and rewards into a single mathematical model for sequential decision making. This framework can represent both episodic tasks (with terminal or absorbing states) and continuing tasks within the same model.
